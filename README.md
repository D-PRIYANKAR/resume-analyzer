# Resume Matcher (NLP Project)

## 🚀 Features
- Extract text from PDF resumes
- Clean text using NLP (spaCy)
- Match resume with job description
- Calculate similarity score
- Detect missing skills

## 🛠 Tech Stack
- Python
- spaCy
- Scikit-learn
- TF-IDF + Cosine Similarity

## ▶️ Run
```bash
python src/main.py
